AAMC-UX-FRAMEWORK

The AAMC UX framework contains a comprehensive guide to styling your apps per the AAMC approved guidelines. It is built on top of Bootstrap 4 with customizations added for branding and approved styling. This project contains the SASS/CSS your application needs. The yeoman generator has already put in the appropriate dependencies so that this is project is pulled in. Please check out the detailed examples and code below

Components

An application is a tree of components: Ideally, the whole application should be a tree of components that implement clearly defined inputs and outputs, and minimize two-way data binding. That way, it's easier to predict when data changes and what the state of a component is.
